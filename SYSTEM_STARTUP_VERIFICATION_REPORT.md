## 第五步：发现的问题与改进建议

### 问题 1：专业名解析覆盖率不足
- 高分段 (rank 5000-10000) 仅有 2-4/45 条结果附带专业名
- 原因：502 条映射未覆盖高分段 985/211 院校的全部专业组
- 建议：补充 985/211 院校的专业组映射

### 问题 2：部分 group_code 含换行符
- 如岭南师范学院的 group_code 为 `广\n202`
- 建议：在数据清洗阶段对 raw source data 进行 clean

### 问题 3：Flutter 前端未启动
- 网络限制导致 `flutter pub get` 失败
- config 中 API baseUrl 为 port 8000，与当前 8002 不匹配
- 建议：本地 IDE 环境下修改端口配置后启动
