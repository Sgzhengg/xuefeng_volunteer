## 🔧 常见问题

### Q1: 端口被占用

**解决方法**：
```bash
# 查看端口占用
netstat -an | findstr :5000
netstat -an | findstr :8080

# 终止进程（替换PID）
taskkill /PID <进程ID> /F
```

### Q2: Python命令不存在

**解决方法**：
1. 安装Python 3.8+
2. 添加到系统PATH
3. 重启终端

### Q3: 前端页面无法加载

**解决方法**：
1. 确保后端服务正在运行
2. 检查浏览器控制台错误
3. 尝试刷新页面

### Q4: 聊天功能无响应

**解决方法**：
1. 检查后端服务：http://localhost:5000/api/v1/health
2. 查看浏览器开发者工具的Network标签
3. 确认API调用正常
