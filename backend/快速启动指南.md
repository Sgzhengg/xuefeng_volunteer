## 🎯 立即开始（3个简单步骤）

### 步骤 1: 创建数据目录

```bash
cd backend
mkdir -p data/cache
```

### 步骤 2: 首次数据更新（可选）

```bash
# 运行管理器
python scraper_manager.py

# 选择选项 1（完整更新）或 2（快速更新）
```

### 步骤 3: 启动后端服务

```bash
# 启动 FastAPI 后端
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**访问地址**：
- API 文档: http://localhost:8000/docs
- 健康检查: http://localhost:8000/health
- 数据源状态: http://localhost:8000/api/v1/data-source/status

## 📊 API 使用示例

### 1. 获取院校列表

```bash
curl http://localhost:8000/api/v1/sunshine/universities?province=浙江
```

**响应**：
```json
{
  "success": true,
  "data": {
    "universities": [
      {"id": "1", "name": "浙江大学", "province": "浙江", "level": "985"}
    ],
    "total": 50,
    "updated_at": "2026-04-22T10:30:00"
  },
  "source": "sunshine"
}
```

### 2. 获取院校详情

```bash
curl http://localhost:8000/api/v1/sunshine/university/1
```

### 3. 获取录取分数线

```bash
curl "http://localhost:8000/api/v1/sunshine/admission-scores/浙江?year=2025"
```

### 4. 批量爬取

```bash
curl -X POST "http://localhost:8000/api/v1/sunshine/batch-scrape?limit=10&delay=2"
```

## 💾 数据缓存

### 缓存位置
```
backend/data/cache/
├── university_list_all.json
├── university_detail_1.json
├── admission_scores_浙江_2025.json
└── major_list.json
```

### 缓存管理

**查看缓存**：
```bash
ls -lh backend/data/cache/
```

**清除缓存**：
```bash
# 方法 1: 使用管理器
python scraper_manager.py
# 选择选项 5

# 方法 2: 手动删除
rm backend/data/cache/*.json
```

**更新缓存**：
```bash
# 强制重新爬取（忽略缓存）
curl "http://localhost:8000/api/v1/sunshine/universities?use_cache=false"
```

## 📱 集成到前端

### Flutter App

```dart
// 配置后端地址
const String backendUrl = 'http://localhost:8000';

// 获取院校列表
final response = await http.get('$backendUrl/api/v1/sunshine/universities');
final data = json.decode(response.body);

// 数据源状态
final statusResponse = await http.get('$backendUrl/api/v1/data-source/status');
final dataSource = json.decode(statusResponse.body);
// dataSource['data']['primary_source'] == 'sunshine_scraper'
```
