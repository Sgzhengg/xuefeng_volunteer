## 🔧 数据收集工具

### 1. 使用导入脚本手动录入

```bash
cd backend
python scripts/import_admission_data.py
```

选择选项 1 (手动录入)，按提示输入：
- 院校名称
- 院校类型 (985/211/省属重点等)
- 城市
- 专业名称
- 专业代码
- 最低录取位次
- 最低录取分数
- 招生计划数

### 2. 使用CSV批量导入

创建CSV文件，包含列：
```
university_name,university_type,city,major_name,major_code,min_rank,min_score,enrollment_count
中山大学,985,广州,计算机科学与技术,080901,4000,635,120
华南理工大学,985,广州,软件工程,080902,8000,620,100
...
```

然后运行：
```bash
python scripts/import_admission_data.py
选择选项 2 (CSV文件导入)
```

### 3. 使用模板数据测试

```bash
python scripts/import_admission_data.py
选择选项 3 (生成模板数据)
```

这将生成10条测试数据，用于验证算法。

## 🎯 收集建议

### 数据来源渠道
1. **官方渠道**: 广东省教育考试院官网
2. **高校官网**: 各大学招生网历年录取数据
3. **教育平台**: 阳光高考、夸克高考等
4. **历年资料**: 高考志愿填报指南书籍

### 数据质量要求
- ✅ **准确性**: 位次和分数必须准确
- ✅ **完整性**: 每个院校至少5个专业
- ✅ **代表性**: 覆盖热门专业（计算机、金融、临床医学等）
- ✅ **时效性**: 使用最近1-3年数据

### 优先收集专业
1. **计算机类**: 计算机科学与技术、软件工程、人工智能
2. **电子信息类**: 电子信息工程、通信工程
3. **电气类**: 电气工程及其自动化、自动化
4. **经管类**: 金融学、会计学、工商管理
5. **医学类**: 临床医学、口腔医学
6. **语言类**: 英语、汉语言文学
