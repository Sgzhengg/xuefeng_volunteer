# 2025年高考数据批量导入系统

## 📋 系统概述

本系统用于批量导入各省教育考试院发布的高考录取数据，支持自动格式检测、转换和导入。

## 🗂️ 目录结构

```
backend/
├── scripts/
│   ├── import_2025_data.py          # 主导入脚本
│   ├── run_import.bat               # Windows启动脚本
│   └── DATA_IMPORT_README.md        # 本文档
├── data/
│   ├── universities_list.json       # 院校基础数据
│   └── major_rank_data.json         # 录取数据（导入目标）
└── province_data_2025/              # 待导入的省份数据
    ├── 广东/
    ├── 河南/
    ├── 山东/
    └── ...
```

## 📦 数据准备

### 1. 获取数据源

从各省教育考试院官网下载2025年高考录取数据CSV文件：

- **广东省**: https://eea.gd.gov.cn/
- **河南省**: https://www.haeea.cn/
- **山东省**: https://www.sdzk.cn/
- **四川省**: https://www.sceea.cn/

### 2. 文件命名规范

将下载的CSV文件按以下目录结构放置：

```
province_data_2025/
├── 广东/
│   ├── 2025本科院校专业录取数据.csv
│   └── 2025专科院校专业录取数据.csv
├── 河南/
│   └── 2025年普通高校录取数据.csv
├── 山东/
│   └── 2025年夏季高考录取数据.csv
└── ...
```

### 3. CSV文件格式要求

文件应包含以下列（支持不同列名格式）：

| 标准列名 | 支持的别名 | 是否必填 | 说明 |
|---------|-----------|---------|------|
| university_name | 院校名称、学校名称、大学名称、院校 | ✅ | 院校全称 |
| major_name | 专业名称、专业 | ✅ | 专业全称 |
| year | 年份、年 | ✅ | 录取年份（2025） |
| province | 省份、省 | ✅ | 所属省份 |
| min_rank | 最低位次、位次、最低排位 | ✅ | 最低录取位次 |
| min_score | 最低分、分数 | ❌ | 最低录取分数（可选） |

**示例CSV格式：**

```csv
院校名称,专业名称,年份,省份,最低位次,最低分
北京大学,计算机科学与技术,2025,广东,150,695
清华大学,电子信息工程,2025,广东,280,693
```

## 🚀 使用方法

### 方法1: 使用批处理脚本（推荐）

1. 双击运行 `run_import.bat`
2. 脚本会自动检查环境、安装依赖、执行导入

### 方法2: 手动执行

1. 安装依赖：
```bash
pip install requests chardet
```

2. 启动后端服务：
```bash
cd backend
python -m app.main
```

3. 在新终端运行导入脚本：
```bash
cd backend/scripts
python import_2025_data.py
```

## ⚙️ 配置说明

### Admin API配置

在 `import_2025_data.py` 中修改配置：

```python
ADMIN_API_URL = "http://localhost:8000"  # 后端API地址
ADMIN_USERNAME = "admin"                  # 管理员用户名
ADMIN_PASSWORD = "password"               # 管理员密码
```

### 省份格式配置

如需添加新省份格式，在 `PROVINCE_CONFIGS` 中添加配置：

```python
"省份名": {
    "encoding": "gbk",           # 文件编码
    "delimiter": ",",            # 分隔符
    "column_mapping": {          # 列名映射
        "CSV中的列名": "标准列名"
    }
}
```

## 📊 导入流程

1. **环境检查**: 检查Python环境和依赖包
2. **API连接**: 测试Admin API连接状态
3. **文件扫描**: 扫描 `province_data_2025/` 目录下所有CSV文件
4. **格式转换**: 根据省份配置自动转换CSV格式
5. **数据导入**: 调用Admin API导入数据
6. **结果统计**: 生成导入报告和日志

## 🔧 故障排除

### 问题1: 连接API失败

**症状**: `❌ 无法连接到Admin API`

**解决方法**:
1. 检查后端服务是否启动
2. 确认API地址是否正确
3. 验证管理员账号密码

### 问题2: CSV格式错误

**症状**: `CSV文件格式不正确，缺少必要的列`

**解决方法**:
1. 检查CSV文件是否包含必填列
2. 确认列名格式是否在支持列表中
3. 检查文件编码是否正确

### 问题3: 编码问题

**症状**: 文件内容乱码

**解决方法**:
1. 使用记事本打开CSV文件
2. "另存为" 选择UTF-8编码
3. 或在省份配置中指定正确编码

### 问题4: 数据重复

**症状**: 大量"重复记录"

**说明**: 
- 系统会自动去重（院校+专业+年份+省份）
- 已存在的记录会被跳过
- 这是正常现象，不是错误

## 📈 导入优化建议

1. **分批导入**: 大量数据建议分省、分批次导入
2. **数据备份**: 导入前备份现有数据
3. **测试先行**: 先用少量数据测试格式
4. **监控性能**: 导入时监控服务器内存和CPU

## 🎯 支持的数据格式

系统支持以下CSV格式变体：

1. **编码格式**: UTF-8, GBK, GB2312
2. **分隔符**: 逗号(,), 制表符(\t), 分号(;), 竖线(|)
3. **列名格式**: 自动识别多种列名别名
4. **日期格式**: 自动转换年份格式
