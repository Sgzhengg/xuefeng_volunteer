## 系统功能

### 1. 核心采集系统
**文件**: `guangdong_2025_admission_scraper.py`

主要功能：

### 2. API分析工具
**文件**: `api_analyzer.py`

主要功能：
- 使用Playwright分析网站结构
- 自动捕获JSON接口
- 生成API分析报告
- 支持用户手动导航辅助分析

### 3. 数据质量检查工具
**文件**: `data_quality_checker.py`

主要功能：
- 检查模拟数据
- 检查数据完整性
- 验证数据真实性
- 检测格式错误
- 生成质量报告

### 4. 配置系统
**文件**: `scraper_config.py`

主要配置：
- 数据源配置
- 采集参数配置
- 验证规则配置
- 输出格式配置

### 5. 便捷工具
- `start_scraper.bat` - Windows快速启动脚本

## 系统特点

### ✅ 数据真实性保障
1. **多源采集**: 从多个权威网站采集数据
2. **交叉验证**: 同一数据至少2个来源一致才标记为已验证
3. **来源追踪**: 每条数据记录来源和采集时间
4. **不一致报告**: 自动记录数据冲突

### ✅ 技术实现
1. **异步采集**: 使用aiohttp提高采集效率
2. **优先级策略**: 官方 > 高考直通车 > 掌上高考 > 中国教育在线
3. **智能去重**: 同学校+同专业组只保留一条记录
4. **格式标准**: 输出CSV和Excel双格式

### ✅ 质量控制
1. **禁止模拟**: 严格禁止使用AI生成或模拟数据
2. **范围验证**: 分数300-750，排位1-500000
3. **格式检查**: 数据类型和格式验证
4. **完整性检查**: 必填字段验证

## 输出文件说明

采集完成后，在 `output/` 目录生成：

1. **guangdong_2025_admission_*.csv** - 标准CSV格式数据
   - UTF-8-BOM编码，Excel可直接打开
   - 包含所有字段和验证状态

2. **guangdong_2025_admission_*.xlsx** - Excel格式数据
   - 标准Excel格式
   - 方便进一步处理和分析

3. **discrepancy_report_*.csv** - 数据不一致报告
   - 记录多源数据冲突
   - 方便人工核查

4. **summary_report_*.txt** - 采集汇总报告
   - 采集统计数据
   - 数据源分布
   - 验证结果统计
