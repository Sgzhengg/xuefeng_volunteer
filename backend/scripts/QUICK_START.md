# 2025广东高考投档线采集系统 - 快速使用指南

## 系统概述

本系统用于从真实数据源采集2025年广东省高考本科批次投档线，确保数据的真实性和完整性。

## 使用步骤

### 步骤1: 检查数据质量

```bash
cd backend/scripts
python data_quality_checker.py
```

这会生成数据质量报告，帮助了解当前数据状态。

### 步骤2: 分析目标网站API（推荐）

在正式采集前，使用API分析工具找到真实的数据接口：

```bash
python api_analyzer.py
```

按照提示：
1. 选择要分析的网站
2. 在打开的浏览器中手动导航到2025年广东投档线页面
3. 按Enter捕获API请求
4. 查看分析报告并保存

### 步骤3: 配置API端点

根据API分析结果，编辑 `scraper_config.py` 中的API端点：

```python
DATA_SOURCES = {
    'gaokao_cn': {
        'possible_apis': [
            '这里填写找到的真实API地址',
        ]
    }
}
```

### 步骤4: 运行完整采集

```bash
python guangdong_2025_admission_scraper.py
```

系统会自动：
1. 从多个数据源采集数据
2. 进行数据去重
3. 多源数据验证
4. 导出CSV和Excel格式

### 步骤5: 查看结果

采集完成后，查看 `output/` 目录：

- `guangdong_2025_admission_*.csv` - 标准CSV数据
- `guangdong_2025_admission_*.xlsx` - Excel格式
- `discrepancy_report_*.csv` - 数据不一致报告
- `summary_report_*.txt` - 汇总报告

## 快速启动（Windows）

双击运行 `start_scraper.bat`，按照提示操作。

## 常见问题

### Q1: 运行后没有采集到数据？

**A**: 可能的原因：
1. 2025年投档线尚未发布（7月中下旬）
2. API端点需要更新（使用api_analyzer.py分析）
3. 网站结构变化

### Q2: 如何验证采集的数据是真实的？

**A**:
1. 查看 `verified` 字段（True表示多源验证）
2. 查看 `discrepancy_report.csv`（如果有冲突）
3. 与广东省教育考试院官方数据对比

### Q3: 数据量太少怎么办？

**A**:
1. 使用API分析工具找到更多数据源
2. 等待官方发布完整的投档线数据
3. 手动从官网下载Excel文件补充

### Q4: 如何只更新已有数据？

**A**:
```python
# 编辑主程序，只采集特定年份或省份
records = await scraper.scrape('物理')
```
