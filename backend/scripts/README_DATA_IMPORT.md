# 广东省教育考试院官方数据导入指南

## 概述

本指南说明如何从广东省教育考试院下载官方2025年投档线数据并导入到学锋志愿系统。

## 关键原则

| 原则 | 说明 |
|------|------|
| **只信官方** | 只从各省教育考试院下载投档线Excel |
| **不要模拟** | 绝对不创建「一般院校X」这类占位数据 |
| **保留溯源** | 每条数据标记来源和年份 |
| **专业组需展开** | 官方数据是专业组，需从高校官网补充专业映射 |

## 数据下载

### 官方数据源

**广东省教育考试院**: https://eea.gd.gov.cn/

### 下载步骤

1. 访问广东省教育考试院官网
2. 找到「普通高校招生录取」栏目
3. 下载以下文件：
   - `2025年本科普通类投档线.xlsx` (7月发布)
   - `2025年专科普通类投档线.xlsx` (8月发布)

### Excel格式

| 院校代码 | 院校名称 | 专业组代码 | 计划数 | 投档人数 | 最低分 | 最低排位 |
|----------|----------|------------|--------|----------|--------|----------|
| 10561 | 华南理工大学 | 202 | 60 | 60 | 625 | 8500 |
| 10561 | 华南理工大学 | 203 | 40 | 40 | 618 | 10200 |

## 使用流程

### 步骤1：清理现有模拟数据（可选）

如果系统中存在旧的模拟数据，先清理：

```bash
cd backend
python scripts/clean_mock_data.py --dry-run  # 预演模式
python scripts/clean_mock_data.py            # 实际清理
```

**清理效果**：
- 删除 `mock_data: true` 的记录
- 删除占位符院校（如「一般院校X」）
- 移动模拟文件到备份目录

### 步骤2：导入官方数据

```bash
cd backend
python scripts/import_official_guangdong_2025.py <excel文件路径>
```

**示例**：
```bash
python scripts/import_official_guangdong_2025.py ../data/2025年本科普通类投档线.xlsx
```

### 步骤3：验证数据质量

导入完成后，脚本会自动生成质量报告：

```

  涉及院校: 280所

[位次段分布]
  1-5000: 120条
```

## 脚本说明

### import_official_guangdong_2025.py

主导入脚本，功能包括：

1. **解析Excel**：支持多sheet、多种列名格式
2. **数据验证**：自动过滤占位符和异常数据
3. **专业组展开**：根据内置映射将专业组展开为具体专业
4. **数据合并**：智能合并到主数据文件
5. **质量报告**：生成详细的数据质量报告

### clean_mock_data.py

清理脚本，功能包括：

1. **占位符识别**：识别并删除所有占位符记录
2. **模拟文件删除**：移动mock文件到备份目录
3. **清理报告**：记录清理操作详情

### test_import_official_guangdong.py

测试脚本，用于验证导入流程。

## 专业组映射

脚本内置了主要院校的专业组映射，位于 `GROUP_MAJORS_MAPPING` 字典中。

如需添加新映射：

```python
GROUP_MAJORS_MAPPING = {
    "院校代码_专业组代码": ["专业1", "专业2", "专业3"],
    # 示例
    "10558_205": ["计算机科学与技术", "软件工程", "网络工程"],
}
```

## 验收标准

✅ 数据中无「一般院校」等占位符
✅ 每位次段至少有 500 条真实记录
✅ 回溯测试命中率 ≥85%
✅ 所有记录标记 `data_source: "广东省教育考试院_2025"`

## 常见问题

### Q: Excel文件解析失败？

A: 检查列名是否包含关键词：
- 院校代码 / 学校代码 / 代码
- 院校名称 / 学校名称 / 院校
- 专业组代码 / 专业组
- 最低分 / 投档分
- 最低排位 / 排位 / 位次

### Q: 如何批量导入多个文件？

A: 使用循环：
```bash
for file in ../data/2025年*.xlsx; do
    python scripts/import_official_guangdong_2025.py "$file"
done
```

### Q: 如何回滚到导入前的状态？

A: 导入脚本会自动创建备份文件：
```
data/major_rank_data_backup_YYYYMMDD_HHMMSS.json
```

恢复方法：
```bash
cp data/major_rank_data_backup_YYYYMMDD_HHMMSS.json data/major_rank_data.json
```
