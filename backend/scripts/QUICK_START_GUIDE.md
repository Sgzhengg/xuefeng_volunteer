# 快速启动指南

## 立即使用

### 手动模式（推荐）

1. **下载官方Excel文件**
   - 访问：https://eea.gd.gov.cn/
   - 找到：2025-07-19发布的"广东省2025年普通高考本科批次正式投档"
   - 下载：普通类（物理）投档情况.xlsx 和 普通类（历史）投档情况.xlsx

2. **导入数据**
   ```bash
   cd backend/scripts
   python fetch_real_guangdong_2025.py manual <excel文件路径>
   ```

3. **查看结果**
   - 主数据：`output/guangdong_2025_data_*.csv`
   - 质量报告：`output/data_quality_report_*.json`
   - 清理日志：`output/clean_log_*.txt`

### 自动模式

```bash
cd backend/scripts
python fetch_real_guangdong_2025.py auto
```

系统会自动尝试从广东省教育考试院获取数据。

## 系统输出

如果成功获取到3000+条真实数据，系统会输出：

```
================================================================================
数据获取完成！
================================================================================
✓ 成功获取 3000+ 条真实数据
✓ 已通过完整性验证
✓ 已生成所有输出文件
```

如果没有获取到数据，系统会明确报错：

```
================================================================================
FAILED_TO_FETCH_REAL_DATA
================================================================================
```

## 数据验证

系统会自动验证：

- ✅ 至少3000条记录
- ✅ 包含985/211院校
- ✅ 包含民办院校
- ✅ 每条记录都有专业组代码
- ✅ 数据范围：100-750分，排位>0

## 文件说明

- `fetch_real_guangdong_2025.py` - 主系统
- `FINAL_REPORT.md` - 完整报告
- `data_quality_checker.py` - 质量检查工具

## 注意事项

⚠️ **重要**：
- 系统严格遵循真实性约束，绝不生成虚假数据
- 如果官方只发布PDF文件，请先转换为Excel
- 确保Excel文件包含必需的列：院校名称、专业组代码、最低分、最低排位
